========================================
C++ 简介
----------------------------------------

1. C++ 被认为是一种中级语言，它综合了高级语言和低级语言的特点。

2. C++ 完全支持面向对象的程序设计，包括面向对象开发的四大特性：

封装
抽象
继承
多态


3. 标准库
标准的 C++ 由三个重要部分组成：

核心语言，提供了所有构件块，包括变量、数据类型和常量，等等。
C++ 标准库，提供了大量的函数，用于操作文件、字符串等。
标准模板库（STL），提供了大量的方法，用于操作数据结构等。
