(2A)两跳连接: 示例
ssh -N \
  -J wangjl@gate1.picb.ac.cn \
  -L 8000:j3.biomooc.com:8000 \
  -L 8787:h3.biomooc.com:8787 \
  wangjl@10.10.117.157

或者写 ~/.ssh/config: /c/Users/wangj/.ssh/config
Host jump
  HostName gate1.picb.ac.cn
  User wangjl

Host rstudio-h3
  HostName 10.10.117.157
  User wangjl
  ProxyJump jump
  LocalForward 8001 j3.biomooc.com:8000
  LocalForward 8787 h3.biomooc.com:8787

然后: $ ssh -N rstudio-h3
	输入两次密码，跳板机一次，内部主机一次，即可登录。
	注意设置准确 HostName 和 User。
